Common Questions

FAQ

Is Echosight safe to use with Path of Exile?

Yes. Echosight is a pure overlay — it does not read game memory, send keystrokes, modify files, or automate gameplay. It behaves like Discord overlay or a hardware monitor: just visible information layered on top.

What hotkeys does it use?

Defaults: Ctrl+Shift+T show/hide overlay, Ctrl+Shift+I toggle interactive mode, Ctrl+Shift+N advance the next task, Ctrl+Shift+Z undo the last task action, and Ctrl+Shift+Y forward it again. All bindings are user-configurable.

What are background tasks?

Tasks that stay dormant until a trigger task completes — for example, "watch for 4-link drops" activates after you finish Act 2. They appear in a separate section below the main list and don't interrupt main-chain progression.

Which platforms are supported?

Windows (portable .exe), macOS (.dmg, Intel + Apple Silicon), and Linux (AppImage + .deb). Grab the latest build from the GitHub Releases page.
